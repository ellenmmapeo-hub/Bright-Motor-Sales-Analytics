Bright Motors Sales Data Analytics
Prepared by Ematsei Datatrics
Motor  Sales Revenue & Profitability Analysis

PROJECT REVIEW
This project delivers a comprehensive data analytics solution for Bright Motors, examining vehicle sales performance across multiple dimensions — year-on-year trends, brand/model profitability, state-level revenue, and seasonal sales patterns. The goal is to surface actionable insights that help Bright Motors move from a revenue-focused mindset to a profit-driven strategy.
The data was stored in Excel and was turned into a CSV file and injected into the BigQuery SQL platform for transformation
In BigQuery, data was transformed via EDA and SQL to Excel; pivot tables were created to provide comprehensive insights.
A dashboard was created using GoogleLooker/DataStudio.
Slides for the presentation were used from CANVA to design slides that match the theme of analyzing car sales 
Other tools, such as MIRO, were used to design the flow diagram
GanttChart and the Mind Map were also designed using CANVA.
